WORD GAME

A word is shown with the letters scrambled and you are required to enter the original word. The app has six modes, with varying difficulty, and keeps track of your score for each.

This is my most recent and most competent creation. Plain JavaScript is used to randomize the order of the letters and count the score, and jQuery is used to change what is displayed to the user.

WD PAINT

Based on MS Paint, this app allows you to change brush color, size, and shape. There is an undo button, but unfortunately no save button.

Before I learned jQuery I was using the Processing.js library which I learned on Khan Academy to do everything. The app uses a draw function from this library which runs 30 times per second and repeatedly draws the selected shape at the user's current mouse position.

POSTURE CHECK

This timer app will audibly remind you to correct your posture at regular intervals. Simply leave it running in an open browser tab.

I programmed this app with very little knowledge or JavaScript and no knowledge of HTML or CSS. Also using the Processing.js library, this app uses a draw function as a slightly unreliable timer, and draws rectangles and graphical text to display content to the user.
